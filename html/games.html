About the Game

Click on available letters to reveal where they appear in the quote. At any point you can reveal the answer or guess the quote, and if your guess is correct it will reveal the answer. Finally, you can bring up a new board by clicking the new game button.

If you have any favorite quotes that you want added to the game, send the initiative a message and we'll try to add it before too long.
